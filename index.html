Panik - Studio

The all-in-one python game engine

Simple

Simplicity was at the top of our priorities, we wanted panik-studio to be as simple as posible.
The simplistic design and the out of the box game elements make game designing blazing fast and child-easy so that anyone at any age can get started with game dev.

Scalable

We wanted to not olny be restrained by simplicity, we wanted to be able to develop full scale games, without restrictions and with top performance. So what did we do about that? Well, since panik-core is a wrapper for PyGame and OpenGL, you can access every part of those modules for full control over your game.

2D or 3D

Panik - Core offers 2D and 3D rendering.
Pygame is responsible for window managing, UI and 2D rendering where as OpenGL renders 3D
